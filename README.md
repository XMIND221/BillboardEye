# BillboardEye

API Node (Express) + app mobile Expo + maquettes v0.

## Démarrage rapide

```bash
npm install
cp .env.example .env   # si présent — configurer Supabase, etc.
npm run dev
```

Mobile : `npm run mobile:start`
